Brief description of the idea.

Tic-tac- toy game is one of the best games that can explan how Mininmax algorithm works. This game has less moves compared to other games like chess. For 2 players, X and O, who immediately start marking spaces on a 3Ã—3 grid. The winner is the player who wins in putting three of their marks in a horizontal, vertical or diagonal board. It's a set game with a predetermined draw, meaning the better game is played by both teams.

Problem formulation:

  • Initial state: emty 3x3 grid
  • Player(s): X&O , X will move first.
  • Actions(s): put player mark in one of 3x3 grid layout.
  • Result(s,a): output of the action, marked grid.
  • Terminal-Test(s): either X or O win orr non(tie)
  • Utility function(s,p): win (+1), lose (-1), and draw (0) if non (Zero sum ).

Brief description of algorithm.

In Minimax, the players are called Maximizer and Minimizer. The Maximizer is attempting to achieve the highest score possible whereas the Minimizer is trying to do the same to get the lowest score possible. Each condition of the board has a value correlated with it. In a given state, if the maximizer has the upper hand, the score of the board will be a positive value. If the minimizer has the upper hand in that board condition, it must have some negative value. these value are calculated using heuristics equation which differ from game to game.
